# 🌦️ Weather App (Python)

## 📌 Description

This is a simple Weather App built using Python.
It fetches real-time weather information for a given city and displays details like temperature, weather condition, humidity, and more.

This project demonstrates the use of APIs, HTTP requests, and data handling in Python.

---

## 🚀 Features

* 🌍 Get weather of any city
* 🌡️ Shows temperature and weather condition
* 💧 Displays humidity and other details
* ⚡ Fast and simple interface
* 🔄 Real-time data using API

---

## 🛠️ Technologies Used

* Python
* Requests Module
* Weather API (e.g., OpenWeatherMap)

---

## ▶️ How to Use

1. Run the Python script
2. Enter the city name
3. View the weather details instantly

---

## 🔑 API Setup
- Get your API key from OpenWeatherMap
- Replace it in the code

## ⚙️ How It Works

* The user enters a city name
* The program sends a request to the weather API
* The API returns weather data in JSON format
* The program extracts useful information
* Displays it in a readable format

---

## ⚠️ Common Issues

* Invalid city name → Check spelling
* API not working → Check API key
* No internet connection → Ensure connectivity
