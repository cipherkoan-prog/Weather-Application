This a weather a application. User have to enter the city's name then it's tell you the Weather(Tempreture, Humidiy, Description) of your city.
Code Description:
Get your API Key from- https://openweathermap.org/api

requests.get() → fetches data from API

json() → converts response into dictionary

main["temp"] → gets temperature

weather["description"] → gets condition (like cloudy, clear)

Example:
Enter the city name: Kolkata

Weather in Kolkata:
Tempreture: 23.97°C
Humidity: 78%
Descriptin: haze
